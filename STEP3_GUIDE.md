# Step 3: Demo Walkthrough Guide

## Overview
This guide helps you prepare and record the demo walkthrough.

## Pre-Recording Checklist

- [ ] LM Studio is installed and models are downloaded
- [ ] CustomGPT is created and tested
- [ ] GEM is created and tested
- [ ] Test files (text and image) are prepared
- [ ] Screen recording software is ready
- [ ] Google Meet or similar tool is set up

## Demo Scenarios & Recommendations

### Scenario 1: Technical Documentation Analysis
**Use Case**: Analyzing a technical document with both local and cloud models
- Show LM Studio analyzing a document
- Show CustomGPT/GEM analyzing the same document
- Compare results and speed

### Scenario 2: Image Understanding
**Use Case**: Processing images with different models
- Upload image to LM Studio
- Upload same image to CustomGPT/GEM
- Compare understanding and responses

### Scenario 3: Real-time Q&A
**Use Case**: Interactive question-answering session
- Ask questions to LM Studio
- Ask same questions to CustomGPT/GEM
- Show response quality differences

### Scenario 4: Custom Instructions Testing
**Use Case**: Testing custom behavior configurations
- Show how custom instructions affect responses
- Demonstrate domain-specific knowledge
- Show practical application

## Recommended Demo Flow

1. **Introduction** (1-2 min)
   - Brief overview of what you'll demonstrate
   - Explain the tools being used

2. **LM Studio Demo** (5-7 min)
   - Show model selection
   - Demonstrate text file upload and processing
   - Demonstrate image upload and processing
   - Show different prompt types
   - Highlight key observations

3. **CustomGPT/GEM Demo** (5-7 min)
   - Show assistant creation process (brief)
   - Demonstrate document upload
   - Show Q&A session
   - Highlight custom instructions
   - Show practical use cases

4. **Comparison** (2-3 min)
   - Side-by-side comparison
   - Discuss pros and cons
   - Use case recommendations

5. **Conclusion** (1 min)
   - Key takeaways
   - Next steps

## Recording Tips

- Enable transcripts in Google Meet
- Use clear, concise narration
- Show mouse movements and clicks clearly
- Pause between sections for clarity
- Test audio/video quality beforehand
- Keep total length: 15-20 minutes

## Post-Recording Steps

1. **Review Recording**
   - Check audio/video quality
   - Verify all steps are clear
   - Edit if necessary

2. **Download Transcript**
   - Export transcript from Google Meet
   - Review and edit for accuracy

3. **Upload to Google Drive**
   - Create folder: `[YourName]_[YourSurname]`
   - Upload video file
   - Upload transcript file
   - Set permissions: "Anyone at Ciklum with the link can view"

4. **Submit**
   - Get shareable links
   - Submit to Google Form

## Expected Outcomes

- Professional demo video (15-20 minutes)
- Accurate transcript
- Google Drive links ready for submission

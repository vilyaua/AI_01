# Step 2: Custom Assistants Guide

## Overview
This guide helps you create and compare CustomGPT and GEM assistants.

## Part A: CustomGPT (ChatGPT Plus Required)

### Steps
1. **Access CustomGPT Editor**
   - Visit: https://chatgpt.com/gpts/editor
   - Ensure you have ChatGPT Plus subscription

2. **Create CustomGPT**
   - Click "Create" or "New GPT"
   - Configure:
     - Name and description
     - Instructions (custom behavior)
     - Knowledge base (upload document)
     - Capabilities (web browsing, code interpreter, etc.)

3. **Upload Document**
   - Choose a relevant document (e.g., technical spec, company handbook)
   - Upload via Knowledge section

4. **Test CustomGPT**
   - Ask questions related to uploaded content
   - Test various question types
   - Document responses

## Part B: GEM (Gemini)

### Steps
1. **Access Gemini Apps**
   - Visit: https://gemini.google.com/app
   - Sign in with Google account

2. **Create GEM**
   - Click "Create" or "New GEM"
   - Configure:
     - Name and description
     - Instructions (custom behavior)
     - Knowledge base (upload same document as CustomGPT)
     - Additional settings

3. **Upload Document**
   - Upload the same document used for CustomGPT (for fair comparison)

4. **Test GEM**
   - Ask same questions as CustomGPT
   - Test various question types
   - Document responses
